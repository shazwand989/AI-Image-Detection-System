### Installation

1. **Clone or download this repository**

   ```powershell
   pip install -r requirements.txt
   ```

3. **Set up API credentials**:
   - Sign up at [Sightengine](https://dashboard.sightengine.com/signup)

     ```powershell
     Copy-Item .env.example .env
     ```

     ```env
     SIGHTENGINE_API_USER=your_api_user_here
     SIGHTENGINE_API_SECRET=your_api_secret_here
     ```

4. **Run the application**:
   ```powershell
   streamlit run app.py
   ```

5. **Open your browser** to http://localhost:8501

## ğŸ› ï¸ Troubleshooting

### "API credentials not configured"
- Ensure `.env` file exists in the project root
- Check that API credentials are correctly set in `.env`
- Verify no extra spaces or quotes around values

### "Import streamlit could not be resolved"
- Install dependencies: `pip install -r requirements.txt`
- Activate your virtual environment if using one

### "Request timed out"
- Check your internet connection
- Large images may take longer to process
- Try reducing image size before upload

### API Error 403 (Authentication Failed)
- Double-check API credentials in `.env`
- Verify credentials at [Sightengine Dashboard](https://dashboard.sightengine.com/)
- Ensure no extra whitespace in credentials
